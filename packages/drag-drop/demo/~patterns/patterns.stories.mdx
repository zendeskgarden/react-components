# Patterns

## Dropzone Custom Icon

A `Dropzone.Icon` sub-component can be used to indicate actions visually. This example
uses `arrow-retweet-stroke.svg` to indicate a "replace" action when `isHighlighted` is set.

<Canvas>
  <Story
    name="DropzoneCustomIcon"
    args={{ isHighlighted: false, messageLabel: 'Drop to replace' }}
    argTypes={{
      messageLabel: {
        name: 'children',
        control: { type: 'text' },
        table: { category: 'Story' }
      },
      isHighlighted: { control: { type: 'boolean' }, table: { category: 'Story' } }
    }}
  >
    {args => <DropzoneCustomIconStory {...args} />}
  </Story>
</Canvas>
